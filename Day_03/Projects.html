Project


EcoTrack :Waste Management System

EcoTrack is a web-based platform for efficient waste management and sustainability. Users can log waste disposal details, track recycling efforts, and monitor progress through an intuitive dashboard. It employs user inputs and data analytics to provide actionable insights, helping individuals and organizations reduce waste and adopt eco-friendly practices.

Technical Stack

Frontend : HTML, CSS, JavaScript

Backend : DJango

Database : MySQL
